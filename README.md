## ## FT-NIR Analysis of the Scaly clade of *Microgramma*

This repository contains the analysis scripts and data for the study: **"A comparison of interspecific and intraspecific phenotypic variation in spectral signatures of ferns with robust versus uncertain species boundaries"**.

**DOI:** [10.1007/s10265-026-01697-1](https://doi.org/10.1007/s10265-026-01697-1) 

### ### 📂 Repository Structure

* 
**`/Scripts`**: R scripts for processing FT-NIR spectra ($4,000\text{--}8,000\text{ cm}^{-1}$), including SNV transformation and PLS-DA modeling.


* 
**`/Figures`**: Graphical outputs, including PCA and confusion matrices for the eight studied species.


* 
**`/Data`**: Raw and processed spectral data from 94 samples.



### ### ✨ Analysis Highlights

This study demonstrates that FT-NIR spectroscopy is a highly effective tool for fern systematics, achieving an average identification accuracy of **81–100%**.

* 
**Taxonomic Robustness**: Well-defined species like *M. percussa* achieved **100% accuracy**, while morphologically overlapping taxa showed lower rates, highlighting potential hybridization or cryptic variation.


* 
**Frond Dimorphism**: The study is the first to evaluate functionally different photosynthetic organs in ferns. Dimorphic species (e.g., *M. nana*, *M. reptans*) exhibited significantly higher intraspecific spectral variability compared to monomorphic ones.


* 
**Advanced Workflow**: Integration of `Standard Normal Variate (SNV)` for pre-processing and `Partial Least Squares Discriminant Analysis (PLS-DA)` with cross-validation (K-fold and LOOCV).
